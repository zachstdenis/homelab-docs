# Core Documentation

This folder contains the foundational reference documentation for the homelab.

These files describe what the environment is, how it is structured, which major systems exist, and how key network and service components fit together.

## Contents

- [Network Topology](network-topology.md) - high-level layout of the homelab network
- [Hardware](hardware.md) - major physical devices and their roles
- [Services](services.md) - core infrastructure and application services
- [DNS](dns.md) - current DNS design using AdGuard Home and Unbound

## Purpose

The documents in this folder are meant to answer questions like:

- what does the homelab currently consist of?
- how is the network laid out?
- what services run where?
- how does remote access work?
- how is name resolution handled?

## Notes

This folder is intended to stay relatively stable and reference-oriented.

Incident writeups, recovery history, and deployment-specific writeups are tracked elsewhere so the core docs remain easier to navigate.
